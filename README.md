# Hidrate-se (Hydrate-se+) 💧📱

Aplicativo Android para ajudar o usuário a acompanhar a ingestão diária de água, registrar consumos, visualizar histórico e ajustar metas/configurações.

---

## 🎯 Objetivo do Projeto

O **Hidrate-se** foi desenvolvido para apoiar o controle de hidratação diária, permitindo:
- Registrar rapidamente a quantidade de água ingerida (ml)
- Acompanhar o total consumido no dia e a meta
- Visualizar histórico de consumo
- Ajustar preferências/metas nas configurações

---

## 🧰 Tecnologias, Linguagens e Ferramentas

### Linguagens
- **Kotlin** (principal)
- **XML** (layouts)

### Plataforma / IDE
- **Android Studio**
- **Gradle** (build e dependências)

### Principais componentes Android
- **Activities** (ex: HomeActivity, HistoryActivity, SettingsActivity)
- **View Binding** (ex: `ActivityMainBinding`)
- **Intent** (navegação entre telas)

---

## 🧱 Arquitetura do Projeto

O projeto segue uma organização simples e prática para apps Android baseados em **Activities**, com separação por `ui` e features (telas/módulos).
